# Day 20 - Student Management System

## Project Overview

A Python-based Student Management System that stores student records permanently using file handling.

## Features

- Add Student
- View Students
- Search Student
- Delete Student
- Persistent Data Storage

## Concepts Used

- Python Functions
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling
- Menu Driven Programs
