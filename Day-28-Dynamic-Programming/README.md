# 🚀 Day 28 – Dynamic Programming (DP)

## 📖 Overview

Today I learned one of the most important Data Structures & Algorithms topics for coding interviews: **Dynamic Programming (DP)**.

Dynamic Programming is an optimization technique that solves complex problems by storing the results of previously solved subproblems, avoiding repeated calculations.
